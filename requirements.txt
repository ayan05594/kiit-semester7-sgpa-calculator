# snap_perf requirements
# 
# This project uses only Python standard library modules.
# Python 3.6+ is required for f-string support and other modern features.
#
# Standard library modules used:
# - argparse (command-line argument parsing)
# - logging (logging functionality)
# - os (operating system interface)
# - re (regular expressions)
# - shlex (shell-like syntax lexical analysis)
# - subprocess (subprocess management)
# - sys (system-specific parameters and functions)
# - datetime (date and time handling)
# - glob (Unix shell-style pathname pattern expansion)
#
# No external dependencies are required.
# 
# Minimum Python version: 3.6
# Recommended Python version: 3.8+
#
# To install Python dependencies (none required for this project):
# pip install -r requirements.txt
#
# Note: The tool requires access to:
# - Performance schedule files (*.cfg)
# - runta interface (external tool)
# - Linux perf or nsys utilities (for profiling)
